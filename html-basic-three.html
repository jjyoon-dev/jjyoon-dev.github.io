HTML(3) - 기본 태그 1

HTML의 기본적인 몇가지 태그에 대해 설명합니다.

HTML 강의 목차

  • HTML(1) - HTML이란?
  • HTML(2) - HTML의 기본구조
  • HTML(3) - 기본 태그 1


<h1>제목</h1>
<p>글 본문</p>
<br/>
<i>글자 기울이기</i>
<b>bold 강조</b>
<strong>strong 강조</strong>
<img src="이미지 경로" />
<a href="">링크</a>
<button>버튼</button>
<ul><li>리스트</li></ul>
<ol><li>리스트</li></ol>
<div></div>
<span></span>

위의 태그들은 <body></body> 태그내에 작성되며 실질적으로 문서를 표현하는데 사용되는 태그들입니다.

1. 헤딩태그 <h1></h1> ~ <h6></h6>

  • 문서내의 각 계층 제목을 나타낼 때 사용합니다.
  • <h1>에서 <h6>으로 갈수록 크기가 작아집니다.
  • 문서의 계층 구조때문에 <h1> 문서 제목은 한번만 사용 하는것이 권장 사항이며, 중간의 번호를 건너뛰고 작성하는 것은 권장하지 않습니다.
예제

See the Pen by jjyoon-dev (@jjyoon-dev) on CodePen.

2.<p></p>

  • Paragraph(문단)의 약자로, 하나의 문단을 생성할때 사용한다.
  • 기본적으로는 ‘enter’를 입력해도 줄바꿈이 일어나지 않으며 단순한 공백(space)처리 됩니다., 줄바꿈시 <br/>태그를 사용해야합니다.

3. <br/>

  • <br/>태그를 삽입하면 해당 위치를 기준으로 줄바꿈이 일어납니다.

4. <i></i>

  • italic의 약자로, 글자를 기울여서 표시합니다.
  • CSS에서 font-style="italic"으로 동일하게 표현할 수 있습니다.

5. <b></b>

  • bold의 약자로, 글자를 굵게 표시하여 강조를 나타냅니다.

6. <strong></strong>

  • bold와 같은 기능을 하지만 최신 표준에선 <strong></strong>을 사용하는 것을 권장합니다.
예제

See the Pen by jjyoon-dev (@jjyoon-dev) on CodePen.

7. <img />

  • 이미지를 삽입하기 위한 태그로입니다.
  • 주요 속성으로 src, alt, width, height 등이 있으며 src=""속성을 통해 이미지 경로를 지정합니다.
  • alt(alternate)속성은 이미지를 불러 올 수 없는 상황이 되었을 때 이미지를 대체하여 alt속성 내의 값을 나타냅니다.
  • width, height는 각각 이미지의 너비와 높이를 지정합니다.
예제

See the Pen by jjyoon-dev (@jjyoon-dev) on CodePen.

8. <button></button>

  • 클릭할 수 있는 버튼을 만들 때 사용합니다.
  • 비슷한<input type="submit/reset/button">과 다르게 button요소 안에는 이미지와 같은 내용을 넣을 수 있습니다.
  • 주의사항으로는 브라우저마다 표현되는 기본값이 다르기 때문에 type속성을 지정해주는것이 좋습니다.

9. <ul><li></li></ul> <ol><li></li></ol>

  • <li></li>태그는 list의 약자로, 목록을 생성하는 태그입니다.
  • 단독으로 사용될 수 없으며 바깥쪽에 <ul></ul>태그 또는 <ol></ol>태그가 감싸주는 형태로 사용되야 합니다.
  • <ul> 태그는 unorderd list의 약자로 리스트에 순서가 없을 경우 사용합니다.
  • <ol> 태그는 orderd list의 약자로 순서가 있는 리스트에 번호를 매길 경우 사용합니다.
  • <ul><ol>태그 하위에 <ul>태그나 <ol>태그를 넣어 이중 리스트를 만들 수 있습니다.
예제

See the Pen by jjyoon-dev (@jjyoon-dev) on CodePen.

10. <div></div>

  • Division의 약자로, 레이아웃을 구분하는데 사용됩니다.
  • 특별한 기능은 없으며 설계의 도구로 사용됩니다. CSS와 함께 사용됩니다.
  • display 속성이 block입니다.
  • display: block속성은 간단하게 요소가 위치한 라인 전체를 차지한다는 것입니다.
  • <div>가 작은 크기의width를 지닌다 하여도 기본적으로 해당 라인은 다른 요소가 침범할 수 없습니다.

11. <span></span>

  • <div></div>태그와 마찬가지로 특별한 기능은 없습니다.
  • <div>와는 다르게 display속성이 inline입니다.
  • display: inline속성은 해당 속성을 지닌 요소들이 라인 하나에 동시에 위치할 수 있습니다.
